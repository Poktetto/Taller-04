Este repositorio de Github contiene el Taller 04 de Programación Orientada a Objetos, en este Taller contiene software con GUI que permite listar, ordenar, modificar y calcular su puntuacion (poder) de la coleccion de cartas Pokemon.

El paquete de este juego es "Taller_04", su clase main es "App.java" y una clase a destacar es que esta construido bajo la Arquitectura de software, por lo tanto su Dominio y Logica estan separados, dentro del Dominio una clase a destacar seria "Carta.java" debido a que es la clase "padre" de todos los demas tipos de Cartas presentes al codigo. Un elemento a destacar en general es el uso de patrones de diseño como el Singleton, Strategy, Visitor y Factory.
